# Architecture Decision Records (ADR)

This directory contains the architectural decisions made throughout the development of QuestForge.

Each ADR documents:

- the context
- the decision
- the alternatives considered
- the consequences

The goal is to preserve architectural knowledge and make technical decisions transparent for current and future contributors.
