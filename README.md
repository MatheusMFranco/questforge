# QuestForge

QuestForge is a gamified productivity platform that combines project management, RPG mechanics, collectible card games (TCG), and Artificial Intelligence to create a more engaging way of managing work.

QuestForge transforms tasks, goals, and team collaboration into an interactive experience.

This repository serves as a long-term engineering project focused on modern software architecture, scalable systems, and AI-powered development.

---

## Goals

- Build a production-grade full-stack application
- Demonstrate modern software engineering practices
- Explore scalable architectures and distributed systems
- Integrate AI agents into real-world workflows
- Serve as a portfolio project for Senior, Staff, and Principal Software Engineer positions

## Architecture

QuestForge follows modern engineering principles, including:

- Clean Architecture
- Domain-Driven Design (DDD)
- SOLID Principles
- Modular Monolith (initially)
- Event-Driven Architecture
- REST APIs
- Microservices (future evolution)

Architectural decisions are documented using **Architecture Decision Records (ADRs)** under the `docs/adr` directory.
