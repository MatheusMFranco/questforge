# QuestForge

QuestForge é uma plataforma gamificada de produtividade.

O objetivo do projeto é servir como estudo de arquitetura moderna, engenharia de software e integração de Inteligência Artificial.

## Stack

- React
- Vite
- TypeScript
- Turborepo
- pnpm
